# Canonical Asset Templates

This directory is the index of templates for canonical asset metadata schema
version `1.0`. Copy the template matching the asset type and replace every
example value; do not introduce a second template family.

| Asset type | Template |
| --- | --- |
| `skill-spec` | `skill-template.yaml` |
| `sub-agent-role-prompt` | `sub-agent-role-prompt-template.yaml` |
| `command-spec` | `command-template.yaml` |
| `prompt-package` | `prompt-package-template.yaml` |

The field contract, types, and enums are defined in
`.ai/assets/CANONICAL-SCHEMA.MD`.

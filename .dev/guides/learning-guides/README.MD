# Learning Guides

本目錄放置 onboarding、learning path、入門導讀型文件。

## 適合放這裡的文件

- 專案學習路徑
- 新成員 onboarding
- 新專案入門指南
- 導讀型文件與閱讀順序建議

## 不適合放這裡的文件

- 實作設定與 troubleshooting
- 架構設計或模型設計參考
- 單一真相規範或 checklist
- AI collaboration / skill / prompt guides
- 單次重構紀錄

這些內容應分別放到：

- `.dev/guides/implementation-guides/`
- `.dev/guides/design-guides/`
- `.dev/standards/`
- `.dev/guides/ai-collaboration-guides/`
- `.dev/refactor-workflows/`

## 文件索引

- `LEARNING-PATH.md`
  - 專案學習路徑
- `NEW-PROJECT-GUIDE.md`
  - 新專案入門與 onboarding 指南

# Guides

This directory contains human-facing guides for AI collaboration, design, implementation, and learning.

## Purpose

Use `.dev/guides/` for durable how-to material, workflow usage notes, onboarding guides, and reference guides intended for human maintainers.

## Usage

Start with `.dev/guides/INDEX.MD` when you need a specific guide or subdirectory. Keep this README focused on the role and boundary of the guides area.

## Boundary

- Put rules, checklists, and source-of-truth standards under `.dev/standards/`.
- Put one-time workflow records under `.dev/workflows/`.
- Put canonical agent-facing assets under `.ai/`.
- Put human-facing skill, prompt, and workflow usage guides under `ai-collaboration-guides/`.

# Durable Assessments

This directory stores durable audits, large code reviews, architecture
assessments, and similar evidence-backed observations that may or may not become
future work.

Use [INDEX.MD](INDEX.MD) to discover assessments. Follow
[`ASSESSMENT-ARTIFACT-POLICY.md`](../standards/ASSESSMENT-ARTIFACT-POLICY.md) for
identity, metadata, branch, immutability, Git lookup, and workflow handoff rules.

Assessments are not workflows and do not authorize remediation. Promote selected
findings to `.dev/backlog/` or reference them from a newly authorized workflow.
